# Настройка аналитики и поисковиков

Код аналитики уже добавлен в `src/layouts/Layout.astro` с placeholder-ами. Вам нужно заменить их на реальные коды.

---

## 1. Яндекс.Метрика

### Создание счётчика

1. Откройте [metrika.yandex.ru](https://metrika.yandex.ru)
2. Войдите в Яндекс-аккаунт (или создайте новый)
3. Нажмите **«Добавить счётчик»**
4. Заполните:
   - **Имя:** Блог Карповой Ольги (или любое)
   - **Адрес сайта:** `litfsing-jpg.github.io/astra/`
   - Включите **Вебвизор**, **Карта кликов**, **Карта скроллинга**
5. Нажмите **«Создать счётчик»**
6. Вы получите **номер счётчика** (8-значное число, например `12345678`)

### Вставка кода

Откройте файл `src/layouts/Layout.astro` и замените `XXXXXXXX` на ваш номер счётчика. Замените в **двух** местах:

- Строка с `ym(XXXXXXXX, "init"` — скрипт инициализации
- Строка с `mc.yandex.ru/watch/XXXXXXXX` — noscript-пиксель

Пример: если номер `12345678`, то:
```
ym(12345678, "init", {
```
и
```
<img src="https://mc.yandex.ru/watch/12345678" ...
```

## 2. Google Search Console

### Регистрация сайта

1. Откройте [search.google.com/search-console](https://search.google.com/search-console)
2. Войдите в Google-аккаунт
3. Нажмите **«Добавить ресурс»** → выберите **«Префикс URL-адреса»**
4. Введите: `https://litfsing-jpg.github.io/astra/`
5. Выберите способ подтверждения: **Тег HTML**
6. Google даст мета-тег вида:
   ```
   <meta name="google-site-verification" content="abc123XYZ..." />
   ```
7. Скопируйте значение `content` (только текст между кавычками)

### Вставка кода

В файле `src/layouts/Layout.astro` найдите строку:
```html
<meta name="google-site-verification" content="[ВСТАВИТЬ_КОД_GOOGLE]" />
```

Замените `[ВСТАВИТЬ_КОД_GOOGLE]` на скопированное значение.

### Отправка Sitemap

1. После подтверждения сайта зайдите в Google Search Console
2. Слева выберите **«Файлы Sitemap»**
3. Введите: `sitemap-index.xml`
4. Нажмите **«Отправить»**

## 3. Яндекс.Вебмастер

### Регистрация сайта

1. Откройте [webmaster.yandex.ru](https://webmaster.yandex.ru)
2. Войдите в Яндекс-аккаунт
3. Нажмите **«Добавить сайт»**
4. Введите: `https://litfsing-jpg.github.io/astra/`
5. Выберите способ подтверждения: **Мета-тег**
6. Яндекс даст мета-тег вида:
   ```
   <meta name="yandex-verification" content="abc123def456" />
   ```
7. Скопируйте значение `content`

### Вставка кода

В файле `src/layouts/Layout.astro` найдите строку:
```html
<meta name="yandex-verification" content="[ВСТАВИТЬ_КОД_ЯНДЕКС]" />
```

Замените `[ВСТАВИТЬ_КОД_ЯНДЕКС]` на скопированное значение.

### Отправка Sitemap

1. После подтверждения зайдите в Яндекс.Вебмастер
2. Выберите ваш сайт → **«Индексирование»** → **«Файлы Sitemap»**
3. Добавьте: `https://litfsing-jpg.github.io/astra/sitemap-index.xml`
4. Нажмите **«Добавить»**

---

## 4. После вставки кодов

1. Сохраните `src/layouts/Layout.astro`
2. Запустите сборку для проверки:
   ```
   npm run build
   ```
3. Запушьте изменения (Obsidian: Ctrl+P → «Obsidian Git: Commit all changes and push»)
4. Подождите 1-2 минуты, пока GitHub Pages обновится
5. Вернитесь в Google Search Console / Яндекс.Вебмастер и нажмите **«Подтвердить»**
